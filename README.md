# PMF(Probabilistic Matrix Factorization)
文中作者提到，传统的协同过滤算法有两个不足：

1).不能很好地处理规模非常大的数据

2). 不能很好地处理那些只给出极少评分的用户

概率矩阵分解则能很好的解决上述提到的这两个问题。
